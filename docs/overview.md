# Overview

This repository is the benchmarking + profiling + cost-effectiveness pillar of a broader edge-AI systems portfolio.

It provides a practical engineering workflow for evaluating heterogeneous edge platforms and runtime stacks under deployment-relevant constraints:

- latency and throughput
- power and thermal behavior
- cost-performance trade-offs
- software backend effects
- concurrent workload allocation behavior

The artifact emphasizes reproducibility and interpretable outputs, not just peak benchmark values.
